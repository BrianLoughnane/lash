# Building Spaces

Building Spaces allows you to segment your buildings into separately tracked areas for a more granular understanding of your buildings' performance.

#### 👍 This article will help you:

* Add spaces to a building
* Assign points to a building's spaces

To access Building Spaces, go to Buildings on the left side menu, select a building, then select the Spaces tab.

#### Add a building space

#### 🚧 Prerequisite: Total area and floors

Before you begin adding spaces, make sure the building has Total area, Top floor, and Ground floor specified in the building's Profile.

If you don't have the exact floor area of your building, you can approximate floor area using estimates from U.S. Department of Energy's Commercial Reference Buildings.

To add a space to a building, select the 'Add a space' button from the building's Spaces tab.

| Field | Description |
| --- | --- |
| Space name | The space's name. Choose a name that best describes the space, like a suite or floor. |
| Space use type | The space's type, such as lobby, office, dining, or retail. |
| Area | The space's area in the building. |
| Floor | The space's floor in the building. |
| Operating hours | The average daily occupancy schedule for this space, from Monday to Sunday. |
| Vendor space ID (optional) | Vendor space ID is a unique identifier that links a space from a third-party system integration to a building in Atrius. |

#### Add a space assignment

To assign a point to a space, select the space's dropdown menu in the table, then 'Edit assignments'.

On the Assignments page, select 'Add an assignment'.

In the Add assignment window, select the point type that you would like to allocate to the space. Next, choose a calculation method:

* Select Choose one point if you have one sub-point for this space.
* Select Estimate from whole building if you do not have submetering for this space, for instance. You may either specify the percent allocation of the point to this space, or select the checkbox to allocate relative to the space's area.
* Select Sum multiple points if you have multiple sub-points for this space.

#### 📘 Using spaces with building tenants

Once you have created a space and assigned points to the space, you can assign tenants to the space. Tenants can be displayed in dashboards and apps.
