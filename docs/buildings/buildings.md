# Buildings

Learn how to get started with buildings in Atrius.

Updated over a year ago

#### 👍 This article will help you:

* Add a building to Atrius
* View building details and access building-specific data

To access Buildings, go to Buildings on the left side menu.

#### Buildings table

The main Buildings page displays a table of all buildings in your organization. Use the Search or Filter to narrow your selection of buildings to view or edit.

#### Add a new building

To add a new building, select the 'Add a building' button. 

Provide all required information on the Building Profile form. Required fields include building name, address, and basic building information. When finished, select the 'Create building' button.

#### Building navigation

Once you've created buildings, you can navigate between them using the Buildings menu or by selecting specific buildings from the main Buildings table.

Each building has dedicated tabs for:
- Overview
- Points
- Bills  
- Weather
- Tenants
- Files
