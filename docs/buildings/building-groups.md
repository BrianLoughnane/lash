# Building Groups

Building Groups allows you to organize your buildings according to type, size, location, ownership, and other parameters unique to your organization.

#### 👍 This article will help you:

* Create building groups to represent different combinations of buildings
* Filter by building group in tables, apps, and cards

To access Building Groups, go to Settings on the left side menu, then Building Groups.

#### Add a building group category

Before you add your first building group, select the 'Add a category' button.

In the modal window, provide a category name, then select 'Create category'. Your custom category will appear below the default Type category. Continue creating building group categories as needed.

"Type" is the default building group category provided by Atrius. It is uneditable. When you define Building type in a building's Profile tab, the building will be grouped with other buildings of the same type. Thus, building groups are automatically created for each type.

#### Add a building group

To add a building group, expand a category to reveal the 'Add a group' button.

In the modal window, provide a group name, then select 'Create group'. Continue creating building groups as needed. To edit an existing building group, select the edit ("pencil") icon.

#### Assign buildings to building groups

After you have created building groups, the next step is to assign buildings to those groups.

There are two ways to assign buildings:

1. Go to the individual building's Profile tab. In the Building Groups section, all the available building group categories will display. Select the building group(s) to which the building belongs. If the building does not belong to a building group, then omit a selection for that building group category.
2. Select the Assignments tab to view a table of all your buildings and their current building group assignments across all the available building group categories. Choose the group in each dropdown menu that applies to that building. If the building does not belong to a building group, then omit selection for that building group category. When finished, press 'Save'.

#### Filter by building group

Look for the Filter in tables throughout Atrius to display buildings and points according to building group and other filter parameters.

Building groups are also useful in apps and dashboards. To create a card for a building group, select a portfolio-scope card, then Building(s) > Building group.
