# Standard dashboards

Standard dashboards are pre-configured, uneditable dashboards that are automatically created by Atrius for your portfolio and every building, tenant, and project.

**Dashboard scope**

A dashboard's scope determines where in Atrius it will be revealed. Standard dashboards are automatically created for each of the following scopes:

- Portfolio: The dashboard will appear on the Atrius homepage.
- Building or Tenant: The dashboard will appear for a specific building or tenant.
- Project: The dashboard will appear for a specific project in a building.

**Portfolio**

The portfolio-scope standard dashboard appears on the Atrius homepage, titled "[Your Organization Name] Overview". The following cards are included:

- Building Summary card displaying consumption/production for each point type across all buildings over the last 12 months
- Portfolio Comparisons card displaying electricity use across all building types over the last 12 months
- Portfolio Trends cards displaying consumption/production for each point type across all buildings over the last 12 months

The dashboard shows available data from points with Scope set to Whole building. If a building has both a Whole building point and a Whole building bill point for the selected point type, the point will be used.

**Changing the default dashboard for the portfolio**

To set a standard or custom dashboard to be the default dashboard on the Atrius homepage, go to Settings > Organization Settings > Homepage default dashboard.

**Building**

The building-scope standard dashboard appears on a building's Dashboards tab, titled "[Your Building Name] Overview". The following cards are included:

- Building Profile card displaying building profile information
- Building Summary card displaying consumption/production for each Whole building point type in the building over the last 12 months
- Building Trends cards displaying consumption/production for each Whole building point type in the building over the last 12 months

**Changing the default dashboard for a building**

To set a standard or custom dashboard to be the default dashboard on a building's Dashboards tab, go to the building's Profile tab, then select a dashboard from the Default dashboard dropdown menu.

**Tenant**

The tenant-scope standard dashboard appears on a tenant's Dashboards tab, titled "[Your Tenant Name] Overview". The following cards are included:

- Tenant Profile card displaying tenant profile information
- Tenant Summary card displaying consumption/production for each point type over the last 12 months
- Tenant Comparisons card displaying electricity use across each building space occupied by the tenant over the last 12 months
- Tenant Trends card displaying consumption/production for each point type over the last 12 months
- Tenant Cost Table card displaying consumption and cost across each building space occupied by the tenant over the last 12 months

**Changing the default dashboard for a tenant**

To set a standard or custom dashboard to be the default dashboard on a tenant's Dashboards tab, select the 'Edit this tenant' button, then select a dashboard from the Default dashboard dropdown menu.

**Project**

The project-scope standard dashboard appears on a project's Dashboards tab, titled "[Your Project Name] Overview". The following cards are included:

- Project Profile card displaying project profile information
- Building Trends cards displaying consumption/production and the project baseline for each point type
