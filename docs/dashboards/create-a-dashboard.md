# Create a dashboard

##### Dashboards are configurable reports that you can create from data in Atrius. Assemble one or more data-driven "cards" to create a dashboard

Dashboards are configurable reports that you can create from data in Atrius. Assemble one or more data-driven "cards" to create a dashboard.

To access Dashboards, go to Dashboards on the left side menu, or Apps > Dashboards.

**Dashboards table**

The main Dashboards page displays a table of all dashboards created in Atrius. Use the Search or Filter to narrow your selection of dashboards to view or edit, or add a new dashboard.

**Edit multiple dashboards at once**

To edit multiple dashboards, select one or more dashboards using the checkboxes on the left side of the table. Then, go to Actions. Here, choose from the available batch actions, like 'Set view/edit access' or 'Duplicate dashboard(s)'.

**Add a new dashboard**

To start with a blank dashboard canvas, select 'Add a dashboard' at the top right of the Dashboards page.

To change the dashboard title, hover over the title, select it, then begin typing. Select 'Enter' when finished. Alternately, select Actions > Dashboard settings to change the title.

**Add a new card**

To create your first card, select the 'Add a card' button. To add more cards to your dashboard, select the 'Add a card' buttons located at the bottom of each column in your dashboard layout.

Select a card type from the menu, then choose your desired parameters for that card. Select 'Save' when finished. View the Cards Glossary to learn more about different card types and how to configure them.

To edit an existing card, hover over the card, open the dropdown menu at top right, and select 'Edit card'. The same configuration options will appear. Select 'Save' when finished.

**Dashboard layout**

To change the dashboard layout, find the Layout menu located at the top right of the dashboard. Switch between a "freeform" layout to stack your cards from top to bottom or a "fixed rows" layout to align your cards from left to right.

To change the number of columns, select the Columns menu. Choose from five available column options: full width, left sidebar, right sidebar, 1/2 split, and 1/3 split.

To reposition a card in the dashboard, hover over the title of the card, then drag and drop it to a new location.

**Dashboard settings**

To edit the dashboard's visibility and sharing settings, go to Actions > Dashboard settings.

Dashboard settings fields:
- Title: Enter a title.
- Scope: Select 'Portfolio' to have the dashboard appear as an option on the Atrius homepage. Select 'Building', 'Tenant', 'Asset', or 'Project' to have it appear as an option on their respective Dashboards tabs. Select 'Other' if it should not appear in any of these locations.
- View access: Select 'All users', 'Specific users', or 'None' to determine which users in your organization are able to view this dashboard.
- Edit access: Select 'All users', 'Specific users', or 'None' to determine which users in your organization are able to edit this dashboard.
- Public dashboard visibility: When enabled, the dashboard and all included content will be accessible to anybody with the 'Public dashboard direct link'.
- Public dashboard direct link: Use this link to access the dashboard directly in a web browser.

By default, new dashboards are set with scope 'Portfolio'. While a dashboard's scope determines where in Atrius it will be revealed, all dashboards can still be viewed from the Dashboards page.

**Building access determines users' data access**

If a user has access only to specific buildings at the org, then that user will only have access to Building-scoped dashboards for the specific buildings the user has access to. The user will not have access to any Portfolio, Tenant, Project, or Asset scoped dashboards, unless another user manually gives view or edit access to individual dashboards via Dashboard settings.

**How can I share a dashboard?**

You can schedule an email to deliver dashboard PDFs to specific users; create a publicly accessible URL for a dashboard; embed cards in an external website; and download data in CSV, PDF, or image formats.
