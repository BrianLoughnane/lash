# Advanced settings

**Learn more about advanced features and settings in dashboards and cards.**

- **Baselines**: Build statistical models to predict resource use based on weather conditions and occupancy.
- **Calendarization**: Allocate bill cost and consumption data to appropriate calendar months for analysis and reporting.
- **Callout adjustments**: Control the main period, comparison period, and percent difference callouts.
- **Emissions**: Assign emission factors for every point type to ensure accurate emissions reporting.
- **Graph adjustments**: Adjust the graph to end at the last month of complete bill data.
- **Normalization**: Remove abnormal variables from consumption data and estimate consumption under typical conditions.
- **Standard dashboards**: Modify the pre-configured dashboards that are automatically created by Atrius.
- **Timeboxing**: Create pragmatic comparisons between different time periods of the same data series by aligning the main period and comparison period as closely as possible.
- **Total energy calculations**: Display total energy for a building, building group, or all buildings.
