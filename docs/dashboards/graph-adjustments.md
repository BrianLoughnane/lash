# Graph adjustments

##### Graph adjustments provide users with control of how graphs are displayed in cards when there are incomplete bill data present

Graph adjustments provide users with control of how graphs are displayed in cards when there are incomplete bill data present.

Because automated utility data can arrive with a one- or two-month lag, Atrius can display the last 12 months of complete bill point data, to maximize dashboard completeness and accuracy.

Graph adjustments only affect the graph, not the data displayed in the callouts.

**Where are graph adjustments available?**

Graph adjustments are available in the following cards:
- Portfolio Trends
- Portfolio Year-over-Year Trends
- Building Trends
- Building Year-over-Year Trends

**How do graph adjustments work?**

Graph adjustments are available for the following selections:
- Data type = Bill points
- Metric = Consumption / Interval demand / Cost / Emissions
- Period = Last 12 months

Once the required selections have been made, a checkbox titled Adjust graph to end at last month of complete data will appear below Period.

A building-scope card will detect the last month of complete data for the selected bill point, while a portfolio-scope card will detect the last month of complete data for all Whole building bill points of the selected point type.

**Troubleshooting**
- The card is not providing the option to adjust the graph: Make sure that you have selected all of the correct options for data type ('Bills') and period ('Last 12 months').
- No data are appearing on the card when the graph has been adjusted: Make sure that the selected bill point has at least one month of complete data within the last 12 months for the selected calendarization method.
