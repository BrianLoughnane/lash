# Card capabilities

##### This article provides an overview of the card attributes currently supported by our most used cards

This article provides an overview of the card attributes currently supported by our most used cards.

For more information about how these attributes operate, review the Cards glossary.

**Portfolio cards**

Portfolio Trends, Portfolio YOY, Portfolio Comparisons support:
- Point type: Total Energy, All point types (Cost, Emissions), Scope 1, 2, 3, Total Emissions, Tags
- Overlay / Comparison: Previous Period / Same period previous year
- Divided by: Area, Occupancy, Trended point, Tag
- Misc.: Calendarization, Dynamic period (Adjust to last 'month' of complete data)

**Building cards**

Building Trends, Building Trends with Stacked bar, Building YOY Trends, Building Comparisons, Building Readout support:
- Point type: Total Energy, All point types (Cost, Emissions), Scope 1, 2, 3, Total Emissions, Tags, Weather attributes (Temp, CDD/HDD, ...)
- Overlay / Comparison: Previous Period / Same period previous year, Weather attributes, Baselines, Goal line, Trended point, Tags
- Divided by: Area, Occupancy, Days in period, Trended point, Tag
- Normalization: Weather, Occupied area
- Misc.: Calendarization, Dynamic period (Adjust to last 'month' of complete data)

**Tenant cards**

Tenant Trends, Tenant YOY, Tenant Comparisons support:
- Point type: All point types (Cost, Emissions)
- Overlay / Comparison: Previous Period / Same period previous year
- Divided by: Area
