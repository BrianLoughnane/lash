# Internationalization

##### Internationalization and localization allow you to tailor Atrius to your locale

Internationalization and localization allow you to tailor Atrius to your locale.

**Language settings**

Language is determined by the following settings:
- **Organization default language**: The language used by your organization is configured in Settings > Organization Settings > Language.
- **User default language**: Users can override the organization default language by going to Settings > User Profile > Language. This change will only apply to the user.
- **Web browser default language**: If neither of the above settings is known, then the default language will utilize the locale setting from your web browser.

**Supported languages**

Atrius supports Chinese (Simplified), English (UK), English (US), French (Canada), French (France), German, Japanese, Spanish, and Norwegian.

**Units settings**

Units settings determines the default building area and temperature units used across Atrius. "Imperial" or "Metric" units of measure can be configured in Settings > Organization Settings.

**Currency conversions**

Currency conversion rates are collected for all international currencies each day. Daily currency conversions are pulled into Atrius from Open Exchange Rates.

Go to Settings > Currency Conversions to view a table of all rates.

**International currencies in apps**

International currencies are supported in:
- Alerts
- Trend Analysis
- Comparisons
- Projects
- Utility Prices

Each of these apps will display the organization's default currency selected in Settings > Organization Settings > Currency.

**International currencies in cards**

International currencies are supported in portfolio-scope cards, building-scope cards, tenant-scope cards, 2-Point Trends card, and Stacked Breakdown card.

**Cards localization settings**

Dashboards and cards support the following localization settings:
- Custom titles
- Custom units
- Localized dates and times
