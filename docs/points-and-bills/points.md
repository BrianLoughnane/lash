# Points

Manage and add new points for your buildings.

Updated over a year ago

#### 👍 This article will help you:

* Navigate your points to view profiles, alerts, baselines, and more
* Add or edit a single point or multiple points at once
* Import or manually enter point readings

To access Points, go to Points on the left side menu.

#### Points table

The main Points page displays a table of all points created in Atrius. Use the Search or Filter to narrow your selection of points to view or edit, or add a new point.

#### Add new points

**Add a single point**

To add a single point, select the 'Add a point' button.

Select an integration for the point. Then, provide all required information on the Point Profile form. When finished, select the 'Create point' button at the bottom of the page.

**Add multiple points**

To create multiple points at once, go to Apps > Batch Point Creation. Follow the steps to prepare and upload your CSV file, then select the 'Upload CSV' button at the bottom of the page.

#### Edit points

**Edit a single point**

To edit a single point, go to Points.

Use the Search or Filter to narrow your selection to find an individual point. Select the dropdown menu on the right side of the table, then select 'Edit point'.

**Edit multiple points**

To edit multiple points at once, go to Points.

Use the Search or Filter to narrow your selection to a subset of points to edit. Select the desired points using the checkboxes at the left side of the table, then select Actions > Edit points or Actions > Set to set a specific attribute for all selected points.

🚧 **Whole building notifications**

Atrius will provide a notification at the top of each building's Points tab if a change is detected that could affect the accuracy of a Whole building point.

Possible notifications include:

* The building has a point for a particular point type that is not included in the Whole building calculated point for that point type. To resolve, you must update the calculated point's formula to include all relevant sub-points.
* The building has points for a particular point type, but it does not have a Whole building point. To resolve, you must create a new Whole building calculated point for that point type, or update an existing sub-point of that point type to scope 'Whole building'.

#### Import point data

**Batch point data upload**

To upload a CSV file with historical data for multiple points at once, go to Apps > Batch Point Data Upload.

Select 'Upload point data'. Customize the CSV template for your data by selecting a point type, unit, reading type, and date/time format. Then, select 'Download CSV' to retrieve the template and follow the instructions. When finished, upload the completed CSV template. Select 'Save'.

❗️**Proceed carefully**

If the data you upload overlaps with existing data in Atrius, the existing data will be overwritten. It is recommended that you export existing data before overwriting it.

**Manual point data entry**

To manually enter point data, go to Points. Select a point and go to the point's Data Manager tab.

In Data Manager, select 'Add a reading'.

Enter the start and end period, and a value. Select 'Save'.

For Manual Points, you can also enter daily totalizer or interval consumption readings via a mobile device.

To access Mobile Data Entry, go to Apps in the side menu at left, then Mobile Data Entry. Alternately, type m.buildingos.com into the web browser on your mobile device.
