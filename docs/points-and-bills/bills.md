# Bills

Manage and add new bill points for your buildings.

Updated over a year ago

#### 👍 This article will help you:

* Navigate your bill points to view profiles, alerts, baselines, and more
* Add or edit a single bill point or multiple bill points at once
* Import or manually enter bill point readings

To access Bill Points, go to Bills on the left side menu.

#### Bill points table

The main Bill Points page displays a table of all bill points created in Atrius. Use the Search or Filter to narrow your selection of bill points to view or edit, or add a new bill point.

#### Add new bill points

**Add a single bill point**

To add a single bill point, select the 'Add a bill point' button.

Select an integration for the bill point. Then, provide all required information on the Bill Point Profile form. When finished, select the 'Create bill point' button at the bottom of the page.

**Add multiple bill points**

To create multiple bill points at once, go to Apps > Batch Bill Point Creation. Follow the steps to prepare and upload your CSV file, then select the 'Upload CSV' button at the bottom of the page.

#### Edit bill points

**Edit a single bill point**

To edit a single bill point, go to Bills.

Use the Search or Filter to narrow your selection to find an individual bill point. Select the dropdown menu on the right side of the table, then select 'Edit bill point'.

**Edit multiple bill points**

To edit multiple bill points at once, go to Bills.

Use the Search or Filter to narrow your selection to a subset of bill points to edit. Select the desired bill points using the checkboxes at the left side of the table, then select Actions > Edit bill points or Actions > Set to set a specific attribute for all selected bill points.

🚧 **Whole building notifications**

Atrius will provide a notification at the top of each building's Bill Points tab if a change is detected, such as a meter replacement, that could affect the accuracy of a Whole building bill point.

Possible notifications include:

* The building has a sub-bill point for a particular point type that is not included in the Whole building calculated bill point for that point type. To resolve, you must update the calculated bill point's formula to include all relevant sub-bill points.
* The building has sub-bill points for a particular point type, but it does not have a Whole building bill point. To resolve, you must create a new Whole building calculated bill point for that point type, or update an existing sub-bill point of that point type to scope 'Whole building'.

#### Import bill point data

**Batch bill data upload**

To upload a CSV file with historical data for multiple bill points at once, go to Apps > Batch Bill Data Upload.

Select 'Upload bill data'. Download the CSV template and enter all required columns for the data being uploaded. Follow the instructions provided at the bottom of the modal window. When finished, upload the completed CSV template. Select 'Save'.

**Manual bill point data entry**

To manually enter bill point data, go to Bills. Select a bill point and go to the bill point's Data Manager tab. Manual readings are most commonly provided for Manual Bills and Atrius Bills integrations. They can also be provided for any integration where there is missing data.

In Data Manager, select 'Add a reading'.

Enter the start and end period, and a consumption, peak demand, or cost value. To enter bill itemizations, toggle between 'Total' and 'Itemized' to reveal additional input fields. Select 'Save'.
