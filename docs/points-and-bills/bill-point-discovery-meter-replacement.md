# Bill Point Discovery & Meter Replacement

#### 👍 This article will help you:

* Navigate to bill point discovery to connect new meters
* Perform meter replacements in Atrius

#### Connecting a new bill point

If you have a new utility meter (due to a meter replacement, for instance), take these steps to properly connect your meter to Atrius and configure it with your existing bill points.

**Step 1: Bill point discovery**

* Go to Settings on the left side menu, then select Integrations > Arcadia Bills.
* Select the Bill Point Discovery tab.
* (Optional) Use Search to look up your new connectable bill point, searching by account number or meter number. This point may or may not have the same point number as an associated Historical point. Note: Your search will pull up any connected or unconnected bill points related to your search.
* Find your bill point in the table. In the Building column, select the correct building from the dropdown menu, then select "Connect". If you have multiple bill points to connect, select one or more points using the checkboxes on the left side of the table. Then, go to Actions > Connect multiple from catalogs.

**Step 2: Configure your bill point**

* From the Bill Point Discovery tab, select the resource type of your newly connected bill point. Doing so will take you to the bill point's Profile page.
* If this is a meter replacement, you will want to configure all fields to match the original point. Next, ensure you mark the original bill point as historical in the point profile. Finally, ensure you are adding the newly created bill point to all calculated points which included the original (now historical) point.
* If this is a new meter and you are unsure how to configure the point, please leave all fields as they are, and contact Atrius Customer Support.

**Step 3: Configure your building**

Every building should have one Whole building point for each point type. Whole building scope tells Atrius which point to use in dashboards and apps to view total resource consumption/production or cost for each resource. When new bill points are connected, they are not automatically added to the Whole building bill point.
