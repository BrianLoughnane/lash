# Data Processing Jobs

##### View the status of running and completed data processing jobs for a point or bill point

View the status of running and completed data processing jobs for a point or bill point.

Data processing jobs are a collection of one or more server-side tasks that reprocess data for a point or bill point. Jobs are either manually initiated by a user or automatically initiated by Atrius.

To access Data Processing Jobs, go to Settings on the left side menu, then Data Processing Jobs.

**View all jobs in your organization**

To view all running and completed data processing jobs in your organization, go to Settings > Data Processing Jobs.

- Select the 'Initiated By Me' tab to view data processing jobs that only you had initiated.
- Select the 'For This Organization' tab to view all data processing jobs across your organization, including jobs that had been automatically initiated by Atrius.

The Data Processing Jobs table allows you to filter by Status, Integration, and User, and to select a time period.

**View jobs for a specific point or bill point**

To view all running and completed data processing jobs for a specific point or bill point, go to Points or Bills on the left side menu. Select a point or bill point from the table, then go to the Jobs tab.

The Data Processing Jobs table allows you to filter by Status, Integration, and User, and to select a time period. If there are any actively running jobs for the point or bill point, a green banner will display at the top of the page.

**Job Status descriptions:**
- Completed: The job was successfully completed.
- Running: The job is actively running.
- Stale: The job has taken too long to complete and is no longer actively running. It is recommended that you initiate the job again or contact Customer Support.
- Failed: The job has failed and was not completed. Please contact Customer Support if a job has failed unexpectedly.
- Canceled: The job was canceled and was not completed.

**Contact Customer Support**

If you have questions about data processing jobs, please contact Customer Support.
