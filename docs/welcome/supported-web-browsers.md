# Supported web browsers

Atrius is designed to work with current web browser technology. We recommend using the latest browser version whenever possible. We currently test against the following browsers:

**Fully supported browsers:**
- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Apple Safari

**Minimum browser requirements:**
- Chrome 88+
- Firefox 85+
- Edge 88+
- Safari 14+

**Mobile browsers:**
- iOS Safari 14+
- Chrome for Android 88+
